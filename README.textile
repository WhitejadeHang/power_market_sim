h1. Simpower - 电力系统优化与市场仿真平台

h2. 基于Python的专业电力系统优化和电力市场仿真平台

Simpower是一个功能完整、经过充分测试的电力系统优化和电力市场仿真平台，基于Pyomo数学优化建模框架，专为电力系统研究人员、工程师和学生设计。

h3. 🌟 核心功能 (代码验证)

* *优化问题求解* (基于 simpower/solve.py):
    - 经济调度 (ED) - 单时段最小成本发电调度 (测试案例: simpower/tests/ed/)
    - 机组组合 (UC) - 多时段混合整数规划优化 (测试案例: simpower/tests/uc/, uc-WW-5-2/)
    - 最优潮流 (OPF) - 网络约束下的DC潮流优化 (测试案例: simpower/tests/opf/)
    - 随机优化 - 考虑不确定性的两阶段随机规划 (测试案例: simpower/tests/stochastic_*/)

* *节点边际电价 (LMP) 计算* (基于 simpower/powersystems.py:167):
    - 完整的对偶变量支持 (simpower/optimization.py:163)
    - 阻塞价格计算 (simpower/powersystems.py:122)
    - 实时电价分析和市场仿真
    - 测试验证 (simpower/tests/test_uc.py:10-22, test_duals.py)

* *电力系统建模* (基于实际类定义):
    - 发电机建模: 20个参数，支持成本曲线、启停约束、爬坡限制 (simpower/generators.py:40-65)
    - 负载建模: 时间序列负载、切负荷功能 (simpower/powersystems.py:35-45)
    - 网络建模: 线路约束、导纳矩阵、DC潮流 (simpower/powersystems.py:104-115)

* *求解器支持* (基于测试验证):
    - 开源求解器: CBC (默认), GLPK (simpower/tests/test_solvers.py)
    - 商业求解器: CPLEX, Gurobi (需许可证)
    - 多求解器兼容性测试 (simpower/tests/test_solvers.py:68-89)
