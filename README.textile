h1. Simpower - 电力系统优化与市场仿真平台

h2. 使用Python的电力系统优化和电力市场仿真

Simpower是一个强大的电力系统优化和电力市场仿真平台，专为研究人员、工程师和学生设计。

h3. 🌟 核心功能

* *优化问题求解*:
    - 经济调度 (Economic Dispatch) - 最小成本发电调度
    - 最优潮流 (Optimal Power Flow) - 网络约束下的优化调度
    - 机组组合 (Unit Commitment) - 考虑启停约束的多时段优化
    - 随机机组组合 (Stochastic Unit Commitment) - 考虑不确定性的鲁棒优化

* *电力市场仿真*:
    - 节点边际电价 (LMP) 计算 - 实时电价分析
    - 现货市场仿真 - 投标、出清、定价机制
    - 备用市场 - 备用容量的定价和调度
    - 阻塞管理 - 输电阻塞的经济分析

* *系统建模*:
    - 发电机组建模 (火电、燃气、风电、光伏等)
    - 负载建模 (时变负载、不确定性建模)
    - 电网建模 (传输线路、变电站、网络约束)

* *求解器支持*:
    - 开源求解器: CBC, GLPK (免费使用)
    - 商业求解器: CPLEX, Gurobi (如已安装)
    - 完整的对偶变量支持 (节点电价计算)
