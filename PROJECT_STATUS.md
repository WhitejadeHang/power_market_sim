## 🔧 技术架构

### 核心模块
```
simpower/
├── powersystems.py     # 电力系统建模 (机组、负载、网络)
├── optimization.py     # 优化引擎 (Pyomo集成)
├── config.py          # 配置管理
├── generators.py      # 发电机组建模
├── loads.py          # 负载建模
├── lines.py          # 输电线路建模
└── tests/            # 测试套件
```

### 求解器支持
- **开源求解器**: CBC, GLPK (已测试验证)
- **商业求解器**: CPLEX, Gurobi (兼容支持)
- **对偶变量**: 完整支持，用于LMP计算

### 优化问题类型
1. **经济调度 (ED)** - 最小成本发电调度
2. **机组组合 (UC)** - 考虑启停约束的多时段优化
3. **最优潮流 (OPF)** - 网络约束下的优化调度
4. **随机机组组合 (SCUC)** - 不确定性下的鲁棒优化
