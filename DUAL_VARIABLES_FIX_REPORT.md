# 对偶变量(节点电价)功能修复报告

## 🎯 修复目标
修复 Simpower 中的对偶变量功能，使其能够正确计算和返回节点边际电价(LMP - Locational Marginal Price)，这是电力市场仿真的核心功能。

## 🔍 问题诊断

### 根本原因
1. **配置问题**: 默认配置中 `duals = False`，导致对偶变量功能被禁用
2. **求解器设置**: 虽然CBC和GLPK支持对偶变量，但需要正确的配置和参数传递
3. **MIP到LP转换**: 机组组合是混合整数问题，需要先求解MIP再固定整数变量重新求解LP获取对偶变量

### 技术细节
- 对偶变量获取需要 `user_config.duals = True`
- 求解器需要 `suffixes=['dual']` 参数
- 约束需要添加 `dual` suffix 来接收对偶变量值
- 机组组合问题需要二阶段求解: MIP求解 → 固定整数变量 → LP重新求解获取对偶变量
