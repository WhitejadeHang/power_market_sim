### **2. 格式对比**

#### **传统格式 (累计总成本)**
```csv
power,cost
0,0
100,1000      # 0-100MW累计总成本1000$
200,2800      # 0-200MW累计总成本2800$
300,5500      # 0-300MW累计总成本5500$
```

#### **新增格式 (容量段报价)** ⭐
```csv
power,price
0,0
100,12.0      # 0-100MW段价格12.0$/MWh
200,18.0      # 100-200MW段价格18.0$/MWh  
300,25.0      # 200-300MW段价格25.0$/MWh
```

### **3. 转换机制**

#### **容量段报价 → 累计总成本**
```python
# 示例转换
段1: 0-100MW @ 12.0$/MWh → 累计成本 = 100 × 12.0 = 1200$
段2: 100-200MW @ 18.0$/MWh → 累计成本 = 1200 + 100 × 18.0 = 3000$
段3: 200-300MW @ 25.0$/MWh → 累计成本 = 3000 + 100 × 25.0 = 5500$
```

#### **累计总成本 → 容量段报价**
```python
# 示例转换
段1: (1000-0)/(100-0) = 10.0$/MWh
段2: (1550-1000)/(150-100) = 11.0$/MWh
段3: (2200-1550)/(200-150) = 13.0$/MWh
```

## 🔧 **使用方法**

### **1. 基础使用**
```python
from simpower.bidding_block import BlockBid

# 创建容量段报价
block_data = pd.DataFrame({
    'power': [0, 100, 200, 300],
    'price': [0, 12.0, 18.0, 25.0]
})

bid = BlockBid(bid_points=block_data, block_format=True)
```

### **2. 工具函数使用**
```python
from simpower.bidding_block import create_block_bid_from_prices

# 从价格列表创建
bid = create_block_bid_from_prices(
    power_points=[0, 100, 200, 300],
    prices=[12.0, 18.0, 25.0]
)
```

### **3. 文件读取**
```python
from simpower.get_data_block import read_block_bid_points

# 自动检测格式并读取
bid_info = read_block_bid_points('generator_bid.csv', format_hint='auto')
```

### **4. CSV格式转换**
```python
from simpower.get_data_block import convert_csv_format

# 转换文件格式
convert_csv_format(
    input_file='block_bid.csv',

```
